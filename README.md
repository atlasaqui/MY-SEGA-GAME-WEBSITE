# 🎧 Jet Set Radio - Modern Web Reimagining

Este projeto é uma reelaboração artística e técnica de como seria o site oficial de Jet Set Radio nos dias atuais.
O objetivo é capturar uma energia semelhante, o estilo graffiti e a trilha sonora icônica do clássico da SEGA, utilizando
tecnologias como React e Tailwind v4.
